CARA PAKAI WEBSITE PT KAYANA

1. Buka index.html untuk halaman utama.
2. Buka hd.html, sedotan.html, gelas.html, atau pp.html untuk katalog produk.
3. Untuk menambah banyak barang, buka file produk-data.js.
4. Copy format data produk, lalu ganti nama, gambar, deskripsi, warna, ukuran, isi, dan kategori.
5. Simpan file gambar produk di folder yang sama dengan HTML.

Contoh kategori:
- hd
- sedotan
- gelas
- pp

Catatan:
File gambar asli tidak ikut lengkap jika belum diupload ke ChatGPT. Pastikan gambar produk disimpan dalam folder website dengan nama yang sama seperti di produk-data.js.
